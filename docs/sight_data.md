# Sight data

##Example of the sight pattern data
```
SSSSSS.##SS..#SS.c.S@SSSS   
```
##Explanation of the symbols
```
## Pattern data symbols
S  Shadow ( area that is out of the sight )
@  Own bot
c  Character ( other bots )
.  Terrain
%  Food
+  Closed door
-  Open door
<  Staircase up    
>  Staircase down  
```
